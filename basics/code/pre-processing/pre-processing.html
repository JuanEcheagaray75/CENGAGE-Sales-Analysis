Una vez que hemos analizado las visualizaciones, notaremos que no hay cambios muy drsticos en la distribucin de las variables, esto es porque originalmente solo hay 9 observaciones de horsepower y 7 de acceleration que tienen como entradas ?, estos representan el 2.26% y el 1.75% de todas las observaciones de la base de datos. El valor que escojamos no afectar enormemente la distribucin general de los datos, aunque por convencin suele usarse la media o la mediana (tcnicas ms especializadas existen).

Para este caso he decidido usar la mediana, ya que es la menos sensible de las 3 a valores atpicos, es fcil de calcular, y funciona bien con bases de datos pequeas (en este caso son 398 observaciones).
